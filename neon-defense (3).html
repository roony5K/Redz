NEON DEFENSE

Cyberpunk Tower Defense

HOW TO PLAY

1. Select a tower type from the bottom panel

2. Click/tap on the grid to place it

3. Press SPACE or tap "Send Wave" to start

4. Enemies follow the dark path - don't let them through!

5. Earn gold from kills to buy more towers

Keyboard Shortcuts:

1-8 Select tower type

ESC Deselect tower

P Pause / Resume

SPACE Send next wave

H Toggle help
